About This Project

A comprehensive machine learning portfolio showcasing various AI capabilities through practical implementations.

Project Overview

This ML Flask Portfolio demonstrates the integration of machine learning models into a web application using Flask. It showcases three distinct ML tasks covering classical machine learning, deep learning, and natural language processing.

Key Features:
  • Classical ML: Iris flower classification using Decision Trees
  • Deep Learning: Handwritten digit recognition with CNNs
  • NLP: Named Entity Recognition and Sentiment Analysis
  • Web Interface: User-friendly Flask web application
  • Real-time Predictions: Instant ML model inference
